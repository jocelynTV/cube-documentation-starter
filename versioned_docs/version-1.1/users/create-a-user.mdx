# Create a user 

Create a user for operation.

## Request

### Endpoint

<Highlight color="#49cc90">POST</Highlight> /api/users

### Body
application/json

- name: `string` `min:6 characters` - User Fullname
- email <sub>required</sub>: `string<email>` - User email address
- password <sub>required</sub>: `string<password>` `>= 8 characters /(?=.*[a-z])(?=.*[A-Z])(?=.*[0-9])/` - User password, MUST contain a mix of upper and lower case letters, as well as digits

### Example

<Tabs>
  <TabItem value="cURL" label="cURL" default>
    ```powershell
    curl --request POST \
      --url http://example.com/api/users \
      --header 'Authorization: Bearer meowmeowmeow' \
      --header 'Content-Type: application/json' \
      --data '{
      "name": "John78",
      "email": "john.smith@example.com",
      "password": "drowssaP123"
    }'
    ```
  </TabItem>
  <TabItem value="Node.js" label="Node.js">
     ```js
    const fetch = require('node-fetch');
    let url = 'http://example.com/api/users';
    let options = {
      method: 'POST',
      headers: {'Content-Type': 'application/json', Authorization: 'Bearer meowmeowmeow'},
      body: '{"name":"John78","email":"john.smith@example.com","password":"drowssaP123"}'
    };
    fetch(url, options)
      .then(res => res.json())
      .then(json => console.log(json))
      .catch(err => console.error('error:' + err));
    ```
  </TabItem>
</Tabs>

## Response
<Highlight color="#49cc90">201</Highlight> Created

### Schema
application/json
```js
{
  "id": "string", // User ID
  "name": "string", // User fullname
  "email": "string" // User email address
}
```
