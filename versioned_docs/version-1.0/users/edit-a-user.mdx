# Edit a user

Edit the details of a user.

## Request

### Endpoint

<Highlight color="#fca130">PATCH</Highlight> /api/users/:user_id

### Path Parameters

`user_id`: `string<uuid>` - ID of the meeting. Fetch the user ID using the create a user API

### Body
application/json

- name: `string` `min:6 characters` - User Fullname
- email <sub>required</sub>: `string<email>` - User email address

### Example

<Tabs>
  <TabItem value="cURL" label="cURL" default>
    ```powershell
    curl --request PATCH \
  --url http://example.com/api/users/92693c4f-c752-4f75-bd44-68dc088a5e26 \
  --header 'Authorization: Bearer meowmeowmeow' \
  --header 'Content-Type: application/json' \
  --data '{
    "username": "John78",
    "email": "john.smith@example.com"
  }'
    ```
  </TabItem>
  <TabItem value="Node.js" label="Node.js">
    ```js
    const fetch = require('node-fetch');

    let url = 'http://example.com/api/users/92693c4f-c752-4f75-bd44-68dc088a5e26';

    let options = {
      method: 'PATCH',
      headers: {'Content-Type': 'application/json', Authorization: 'Bearer meowmeowmeow'},
      body: '{"username":"John78","email":"john.smith@example.com"}'
    };

    fetch(url, options)
      .then(res => res.json())
      .then(json => console.log(json))
      .catch(err => console.error('error:' + err));
    ```
  </TabItem>

## Response
<Highlight color="#49cc90">200</Highlight> OK

### Schema
application/json
```js
{
  "id": "string", // User ID
  "name": "string", // User fullname
  "email": "string" // User email address
}
```
